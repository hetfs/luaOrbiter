   ```bash

   ```bash
   npm install
   ```

4. **Create a feature branch**

   ```bash
   git checkout -b feature/your-topic
   ```

5. **Make your changes** (in `docs/`, `static/`, `src/`, etc.)

6. **Run locally**

   ```bash
   npm run start
   ```

   ```bash
   git add .

   git push origin feature/your-topic
   ```
